Objetivos TT2:
1. Implementar un sistema de almacenamiento de datos que permita resguardar información de manera temporal durante condiciones adversas.
2. Desarrollar un modelo predictivo basado en inteligencia artificial, utilizando datos sintéticos, para anticipar inundaciones a partir de la información recopilada por los sensores.
3. Programar un sistema de predicción de inundaciones que emita alertas automáticas al detectar niveles críticos de agua o riesgos potenciales.
4. Integrar un sistema de comunicación que permita la transmisión de datos entre los sensores loT y el sistema de gestión de datos.
5. Validar la programación del sistema de predicción mediante pruebas controladas con datos simulados para comprobar su funcionalidad.
6. Implementar el sistema de seguimiento solar para el suministro de energía al dispositivo.
7. Implementar y evaluar la base modular diseñada para verificar su estabilidad y funcionalidad durante las pruebas operativas del dispositivo
8. Realizar pruebas en un escenario controlado para evaluar el funcionamiento integral del sistema.
